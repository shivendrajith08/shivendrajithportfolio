## 🌌 Overview

A **world-class, award-level personal portfolio** built entirely in vanilla HTML, CSS, and JavaScript — no frameworks, no build tools, just pure craftsmanship. Designed to impress recruiters, hiring managers, and companies globally.

> *"The kind of portfolio that makes companies stop scrolling and start hiring."*

---

## ✨ Features

### 🎬 Visual & Animation
- **Cinematic loader** — full-name clip-path reveal with staggered entrance animation
- **WebGL 3D starfield** — 3-depth-layer star system with parallax, coloured nebula dust, wireframe geometries (TorusKnot, Octahedron, Icosahedron), and shooting star streaks
- **Mouse-reactive 3D** — entire Three.js scene tilts and parallaxes with cursor movement
- **Hero title reveal** — Bebas Neue typography with line-by-line mask animation
- **Grain/noise texture overlay** — subtle film grain for that premium editorial feel
- **Section scroll reveals** — every element staggers in with cubic-bezier easing

### 🖱️ Interaction
- **Custom morphing cursor** — dot + lagging ring that expands and shows "View" text on hover
- **Magnetic buttons** — CTAs physically attract toward the cursor
- **Horizontal drag-scroll** — project showcase supports mouse drag and touch swipe
- **Radial glow cards** — service cards glow exactly where the mouse pointer is
- **Scroll progress bar** — gradient line tracks reading progress at the top

### 🏗️ Structure & UX
- Floating navbar with blur glassmorphism + scroll-state transformation
- Skills matrix with animated bars triggered on scroll
- Marquee ticker — infinite scrolling tech stack strip
- Horizontal scroll project showcase
- Glowing vertical timeline for experience
- Animated scroll indicator

### 📱 Quality
- Fully **responsive** down to mobile
- `prefers-reduced-motion` respected for accessibility
- **Zero dependencies** — Three.js from CDN, everything else vanilla
- **Single file** — drop `index.html` anywhere and it works
